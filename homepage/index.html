The Java Assembler Interpreter has been developed by 2nd term students for the Department of Technical Informatics at Technical University of Munich. It is designed as a learning tool for x86 assembler. While not implementing all the functionality of a modern Intel Processor its tutorials allow you to try and understand the basic commands and structures of assembler code.
